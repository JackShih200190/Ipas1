```
風險的定義:
是指「威脅(外在)」利用資產的「脆弱性(自己)」直接或間接產生「衝擊」。
```

## 風險處理活動
```
降低(Reduce)
移轉(Transfer) 買保險
接受(Accept)
避免(Avoid)

高可能高衝擊:避免
高可能地衝擊:降低
低可能高衝擊:移轉
低可能低衝擊:接受

殘餘風險<可接受風險
```
